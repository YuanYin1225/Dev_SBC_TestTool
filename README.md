# SBC 电磁阀驱动器测试工具 V3.2

基于 Electron 的图形化串口测试工具，用于 SBC 电磁阀驱动器的通道控制、实时数据监控、指令批量生成与发送。

## 系统要求

- Windows 10 / 11
- Node.js 18+
- 可用 COM 端口（USB 转 RS485，9600/8N1）

## 快速开始

```bash
# 安装依赖（国内镜像）
npm config set registry https://registry.npmmirror.com
set ELECTRON_MIRROR=https://registry.npmmirror.com/-/binary/electron/
npm install

# 启动应用
npm start
```

## 打包发布

```bash
npm run build:win
```

输出 `dist/` 目录，生成 portable 免安装版本。

## 功能说明

### 1. 串口连接

- 自动扫描可用 COM 端口，每 3 秒静默刷新（无变化不重建 DOM）
- 波特率 9600，8 数据位，无校验，1 停止位
- 连接状态指示：绿点 = 已连接，红点 = 已断开

### 2. 通道控制（28 路）

- 左侧面板 J1~J14 分组，每组 K1/K2 两路
- 单击通道按钮切换开关状态（蓝色高亮 = 开启）
- 批量操作：全部开启、全部关闭、仅 1 路、仅 2 路
- 点击 **发送控制指令** 下发 9 字节控制帧到驱动器
- 底部实时预览当前指令 hex

### 3. 实时数据监控

| 显示项 | 含义 | 单位 | 协议缩放 |
|--------|------|------|---------|
| V_12VT | 系统供电电压 | V | 原始值 ×0.1 |
| V_DCF | 电磁阀驱动电压 | V | 原始值 ×0.1 |
| I_DCF | 电磁阀总电流 | mA | 原始值 ×10 |
| I_MCU | MCU 电流 | mA | 原始值 ×10 |

- **查询实时数据** 按钮手动查询
- 勾选「自动轮询」每秒自动查询
- 应答 CRC 校验失败时显示错误提示

### 4. 快捷操作

| 按钮 | 发送指令 | 说明 |
|------|---------|------|
| 自检 | `55 AA 00 07 AF AF 64` | 触发完整自检流程 |
| 所有通道开启1s | `55 AA 00 07 00 01 07` | 全通道脉冲输出 |
| 固件版本 | `2D 2D 2D 2D 2D 2D 2D` | 查询版本号，显示在按钮旁 |
| 故障检测开 | `55 AA 00 07 A0 01 A7` | 开启 KZ 故障灯闪烁 (写 Flash) |
| 故障检测关 | `55 AA 00 07 A0 00 A6` | 关闭 KZ 故障灯闪烁 (写 Flash) |
| 故障灯查询 | `55 AA 00 07 A1 00 A7` | 查询当前开关状态，显示在按钮旁 |
| 设备复位/升级 | `2B 2B 2B 2B 2B 2B 2B` | 进入 Bootloader（有确认弹窗） |

### 5. 手动指令

- 输入 hex 指令（空格分隔），如 `55 AA 00 07 60 00`
- 回车或点击发送

### 6. 指令生成器

穷举 28 通道中任选 N 个的所有组合，生成控制指令：

| N | 组合数 | 生成 | 批量发送 |
|---|--------|------|---------|
| 2 | 378 | ✅ | ✅ |
| 3 | 3,276 | ✅ | ✅ |
| 4 | 20,475 | ✅ | ✅ |
| 5 | 98,280 | ✅ | ✅ |
| 6 | 376,740 | ✅ | ✅ |
| 7~ | 118 万+ | ✅ | ❌ 仅导出 |

- **流式生成**：指令不全部存内存，边生成边写入 Blob 分片，690 万条也不会白屏
- **导出格式**：CSV（含表头，兼容 Excel）或 TXT（纯 hex 指令）
- N≥7 时隐藏「批量发送」按钮，仅支持导出文件

### 7. 批量发送

- **加载 CSV**：支持多文件同时选择，自动解析 hex 指令
- **指令生成器直通**：生成后直接点「批量发送」送入队列
- **发送控制**：开始 / 暂停 / 继续 / 停止
- **间隔可调**：默认 1000ms，范围 100ms ~ 60s
- **进度显示**：实时进度条 + 下条预览
- **日志节流**：批量发送时每 50 条记一条摘要，防止日志刷屏

### 8. 通讯日志

- 右侧面板实时显示收发数据
- TX（绿箭头 →）/ RX（蓝箭头 ←）/ 错误（红色 !）
- 环形缓冲区 300 条，requestAnimationFrame 批量渲染，不卡界面
- 一键清空

## 通讯协议

所有帧遵循 `帧头(3B) + 长度(1B) + 载荷(N B) + 校验和(1B)` 格式。
校验和 = 前面所有字节累加取低 8 位。

### 控制指令（9 字节）

```
55 AA 00 09 [ch3] [ch2] [ch1] [ch0] [CRC]
```

4 字节通道位图，需持续循环发送维持电磁阀开启。

### 查询实时数据

```
发送: 55 AA 00 07 60 00 66
应答: 55 BB 00 0F [ch3] [ch2] [ch1] [ch0] [I_MCU] [I_DCF] [V_12VT] [V_DCF] [V_VCC] [T0] [CRC]
```

- 电压类：原始值 × 0.1 V
- 电流类：原始值 × 10 mA
- T0：通讯超时参数，单位 10ms
- 状态码 0xCC = 接收失败

### 快捷指令速查

| 功能 | 完整帧 |
|------|--------|
| 自检 | `55 AA 00 07 AF AF 64` |
| 全通道脉冲 | `55 AA 00 07 00 01 07` |
| 故障灯开 | `55 AA 00 07 A0 01 A7` |
| 故障灯关 | `55 AA 00 07 A0 00 A6` |
| 故障灯查询 | `55 AA 00 07 A1 00 A7` |
| 固件版本 | `2D 2D 2D 2D 2D 2D 2D` |
| Bootloader | `2B 2B 2B 2B 2B 2B 2B` |

## 技术栈

- **Electron 28** — 跨平台桌面框架
- **serialport 12** — Node.js 串口通讯
- **electron-builder** — 打包分发
- 原生 HTML/CSS/JS，无前端框架依赖

## 性能优化

| 优化项 | 方案 |
|--------|------|
| 日志渲染 | 环形缓冲区 + requestAnimationFrame 批量 innerHTML，最多 300 条 |
| 批量发送日志 | 每 50 条记一条摘要，避免 DOM 洪水 |
| 指令生成 | 流式生成 + Blob 分片，690 万条不爆内存 |
| CSV 下载 | 分 2 万行一片构建 Blob，避免巨型字符串拼接 |
| 串口刷新 | 指纹比对，无变化不重建 DOM，消除闪烁 |
| 串口解析 | 长帧正确识别（length = 整帧长度非载荷长度） |

## 故障排除

1. **串口无法打开** — 检查端口是否被占用、驱动是否安装、尝试管理员权限
2. **查询无响应** — 检查 RS485 接线（A+/B-）、确认设备已上电、观察 CRC 状态
3. **npm install 失败** — 设镜像：`ELECTRON_MIRROR=https://registry.npmmirror.com/-/binary/electron/`
4. **大 N 生成慢** — N≥8 时组合数超百万，生成需 1~2 分钟，属正常现象
